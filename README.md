

# statistics_processing
author: Julio Oliveira

A gRPC service to calculate summary statistics from financial transactions.

## Getting Started

This project requires Docker to be installed.

### Quickstart

Running it then should be as simple as:

```console
$ make build
$ make run
```

### Testing

``make test``
